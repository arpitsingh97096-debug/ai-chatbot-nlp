# 🤖 AI Chatbot (NLP + Machine Learning)

## 📌 Overview
This project is a simple AI-powered chatbot built using Natural Language Processing (NLP) and Machine Learning.  
It can understand user input and respond based on learned intent patterns.

---

## 🚀 Features
- Intent-based conversation system  
- Machine Learning model (Naive Bayes)  
- Handles variations in user input  
- Simple and extendable architecture  

---

## 🛠️ Tech Stack
- Python  
- NLTK  
- Scikit-learn  
- Flask (optional UI)

---

## ▶️ How to Run
1. Clone the repo  
2. Install dependencies: pip install nltk scikit-learn flask
3. Run: python app.py 
---

## 💡 Example
You: hello bro  
Bot: Hi there!
